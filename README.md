【簡単な拡張機能のインストール手順】

1 Download this repo as a ZIP file from GitHub.
<br>
2 Unzip the File（ファイルを解凍する）
<br>
3 Chromeの拡張機能ページを開きます。chrome://extensions/ に移動してください。
<br>
4 解凍したファイルを拡張機能ページの中にドラッグ＆ドロップします。
<br>
5 以上です。これでようやく拡張機能を使えるようになりました！
